# 🖼️ Image Gallery

A modern and responsive image gallery built with HTML, CSS, and JavaScript. The project includes category-based filtering, a lightbox viewer, keyboard navigation, touch swipe support, and smooth hover effects.

## ✨ Features

- 📂 Category Filters
- 🖼️ Lightbox Preview
- ⌨️ Keyboard Navigation
- 📱 Touch Swipe Support
- 🎨 Responsive Design
- ⚡ Lazy Loading Images
- 🌟 Smooth Hover Animations

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
